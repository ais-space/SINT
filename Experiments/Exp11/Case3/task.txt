# Task: Implementation of Universal Basic Income: Consensus-Building Analysis

## Description
A multi-round consensus-building exercise on implementing Universal Basic Income with structured rating systems and synthesis.

## Experts
### Expert 1: Economist
**Position:** Pilot programs with gradual implementation
**Arguments:**
1. Empirical data needed before full implementation
2. Gradual rollout reduces fiscal risk
3. Allows for policy adjustments based on results

**Rating:** 8.0/10

### Expert 2: Social Policy Expert
**Position:** Targeted implementation for specific demographics
**Arguments:**
1. Focus on groups most in need
2. Reduces overall cost while providing benefit
3. Addresses specific social issues

**Rating:** 7.5/10

### Expert 3: Political Scientist
**Position:** Broad-based implementation with political support
**Arguments:**
1. Universal programs build stronger public support
2. Reduces political opposition over time
3. More efficient administrative structure

**Rating:** 7.0/10

## Rating System
### Criteria
1. Fiscal Sustainability: 1-10 rating
